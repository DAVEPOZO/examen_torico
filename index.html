50 Términos Fundamentales en Desarrollo Web

Nº Término Definición
1HTMLLenguaje de marcado utilizado para estructurar contenido en la web.
2CSSLenguaje que permite aplicar estilos visuales a los elementos HTML.
3JavaScriptLenguaje de programación que permite agregar interactividad a las páginas web.
4DOMModelo de objetos que representa la estructura de un documento HTML o XML para manipulación dinámica.
5HTTPProtocolo de comunicación utilizado para la transferencia de datos en la web.
6HTTPSVersión segura de HTTP que utiliza cifrado SSL/TLS para proteger la transferencia de datos.
7ServidorEquipo o software que proporciona recursos o servicios a otros dispositivos en la red.
8ClienteDispositivo o programa que solicita recursos o servicios a un servidor.
9URLLocalizador uniforme de recursos que indica la dirección específica de una página web o recurso.
10DominioNombre legible que identifica una dirección IP en Internet.
11HostNombre del servidor que aloja una página web o recurso en la red.
12DNSSistema que traduce nombres de dominio legibles a direcciones IP numéricas.
13POSTMétodo HTTP que envía datos al servidor para crear o actualizar recursos.
14GETMétodo HTTP para solicitar datos de un recurso específico desde un servidor.
15APIInterfaz que permite la comunicación entre diferentes programas o servicios.
16JSONFormato ligero de intercambio de datos basado en texto y fácil de leer.
17XMLLenguaje de marcado para estructurar y transportar datos de manera jerárquica.
18CookiePequeño archivo que almacena información en el navegador para mantener estados entre sesiones.
19SessionMecanismo para mantener datos temporales en el servidor sobre la interacción de un usuario.
20FrameworkConjunto de herramientas y librerías que facilitan el desarrollo de aplicaciones web.
21BootstrapFramework CSS que permite crear diseños responsivos y estilizados rápidamente.
22Responsive DesignDiseño web que se adapta automáticamente a diferentes tamaños de pantalla y dispositivos.
23PHPLenguaje de programación del lado servidor para crear contenido web dinámico.
24MySQLSistema de gestión de bases de datos relacionales utilizado en aplicaciones web.
25SQLLenguaje de consulta estructurado para administrar y manipular bases de datos.
26AJAXTécnica para actualizar partes de una página web sin recargarla completamente.
27JSON Web Token (JWT)Método compacto y seguro para transmitir información de identidad entre partes.
28RESTArquitectura para diseñar servicios web que usan HTTP para operaciones CRUD.
29API RESTfulAPI que sigue los principios REST para permitir comunicación simple y escalable.
30CacheAlmacenamiento temporal de datos para acelerar el acceso en futuras solicitudes.
31SSL/TLSProtocolos que aseguran la comunicación cifrada entre cliente y servidor.
32DOM ManipulationProceso de modificar dinámicamente la estructura o contenido de una página web mediante scripts.
33ViewportÁrea visible de una página web en un dispositivo, crucial para diseño responsivo.
34Web ServerSoftware que responde a solicitudes HTTP para entregar contenido web.
35CMSSistema de gestión de contenido que facilita la creación y administración de sitios web.
36Bootstrap GridSistema de diseño basado en filas y columnas para crear layouts responsivos.
37VariableEspacio de memoria que almacena valores para ser usados y modificados en programación.
38FunctionBloque de código reutilizable que realiza una tarea específica en programación.
39CallbackFunción pasada como argumento para ejecutarse tras la finalización de otra función.
40Event ListenerMecanismo que detecta y responde a eventos de usuario o del sistema en la web.
41Bootstrap ComponentElementos predefinidos de UI disponibles en el framework Bootstrap.
42FlexboxMétodo CSS para diseñar layouts flexibles y alineados de manera sencilla.
43Grid LayoutModelo CSS para crear diseños de página basados en filas y columnas bidimensionales.
44Viewport Meta TagEtiqueta HTML que controla cómo se muestra una página en dispositivos móviles.
45Responsive ImagesTécnicas para adaptar imágenes a distintos tamaños y resoluciones de pantalla.
46Lazy LoadingEstrategia para cargar contenido o imágenes solo cuando son visibles para optimizar rendimiento.
47SEOOptimización para motores de búsqueda que mejora la visibilidad de un sitio web.
48AccessibilityPrácticas para que los sitios web sean utilizables por personas con discapacidades.
49Cross-Origin Resource Sharing (CORS)Política que controla el acceso a recursos web desde dominios diferentes.
50Viewport Width (vw)Unidad relativa en CSS que representa un porcentaje del ancho de la ventana del navegador.
